# AI-Based Resume Screening System

## Overview
This project is an AI-based Resume Screening System that compares candidate resumes with a Job Description (JD) and calculates a match score using Natural Language Processing (NLP).

## Features
- Resume parsing
- Job Description matching
- Match score calculation
- Matching skills extraction
- Missing skills identification
- Resume ranking

## Technologies Used
- Python
- Jupyter Notebook
- pandas
- scikit-learn
- PyPDF2

## Project Structure
- AI-Based Resume Screening System.ipynb
- Resumes/
  - RahulDhope.pdf
  - NikitaBhosale_fresher.pdf
  - NikitaBhosale (2).pdf

## How to Run
1. Install the required libraries.
2. Open the Jupyter Notebook.
3. Add resumes to the `Resumes` folder.
4. Run all cells.
5. Enter a Job Description to compare resumes.

## Output
- Resume Match Score
- Matching Skills
- Missing Skills
- Ranked Resumes
